# Профессиональный фотограф Марина Алехнович

Мой портфолио-сайт и лендинг для фотоуслуг.

## Описание

Сайт создан для демонстрации моих работ фотографа и привлечения новых клиентов. Включает в себя:
- Портфолио работ по категориям
- Интерактивный прайс-лист
- Онлайн-расписание через Calendly
- Корзина услуг с возможностью оформления заказов
- Контактная информация

## Развертывание на GitHub Pages

Сайт настроен для работы с GitHub Pages по адресу: `https://marisol1906.github.io/Photography/`

### Настройки репозитория

1. Перейдите в настройки репозитория (Settings)
2. Выберите раздел "Pages" 
3. Установите источник "Deploy from a branch"
4. Выберите ветку `main` и папку `/ (root)`
5. Сохраните настройки

### Ключевые особенности

- **Base URL**: Настроен для работы с поддоменом `/Photography/`
- **Изображения**: Все ресурсы используют относительные пути
- **Calendly**: Интегрирован виджет для записи на фотосессию
- **Мобильная адаптация**: Полностью адаптивный дизайн
